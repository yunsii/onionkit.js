# 打包发布

- `npm run build`
- `npx changeset`
- `npx changeset version`
- `git commit`
- `npx changeset publish`
- `git push --follow-tags`

> [`changeset` 预发布文档](https://github.com/changesets/changesets/blob/main/docs/prereleases.md)
